ЧТО ЛЕЖИТ В ПАПКЕ
  index.html   — само приложение (открывается в браузере)
  map.webp     — карта (можно заменить на новую)
  config.json  — данные: города, реки, масштаб, маска суша/море
                 (появляется после первого «Экспорта» из редактора)
  README.txt   — этот файл

────────────────────────────────────────────────────────────
1. КАК ПОЛЬЗУЮТСЯ ИГРОКИ (обычная ссылка)
────────────────────────────────────────────────────────────
  • Колесо мыши — зум, перетаскивание — двигать карту, «⟲ Сброс» — вписать.
  • Поиск (вверху): вводят название города — точка подсвечивается на карте,
    карта центрируется. Крестик ✕ очищает и снимает подсветку.
  • Галка «Показывать названия» — подписи городов поверх карты (по умолч. выкл,
    у карты свои подписи).
  • Кнопка ☰ слева вверху — свернуть/развернуть боковую панель.
  • Вкладка «Путь»: кликают по карте, ставят точки A → остановки → Б.
    Клик рядом с городом — точка притягивается к нему.
    Точки можно ПЕРЕТАСКИВАТЬ прямо на карте, в списке слева у каждой есть
    крестик (✕ удалить) и их порядок меняется перетаскиванием строк.
  • Выбирают ПУТНИКА (бюджет): Простолюдин / Купец-аристократ / Гос. посыльный,
    и ТИП маршрута: Авто (самый быстрый), Только посуху, Только по воде.
  • Приложение само считает самый быстрый путь: смешивает сушу, реку и море,
    само определяет вниз/вверх по течению, и расписывает по сегментам —
    «X дней пешком → Y дней вниз по реке → Z дней морем». Цвет линии = способ.
  • Вкладка «Врата»: время телепорта между городами (премиум / эконом).

────────────────────────────────────────────────────────────
2. РЕДАКТОР — ТОЛЬКО У ТЕБЯ
────────────────────────────────────────────────────────────
  Открывается СЕКРЕТНОЙ ссылкой — добавь в конец адреса #editor:
        https://твой-сайт.netlify.app/#editor
  Обычная ссылка (без #editor) редактора НЕ показывает. Юзеры его не видят.
  Сохрани ссылку с #editor себе в закладки.

  ИНСТРУМЕНТЫ (блок «Редактор»):
   • Карта (файл) — подгрузить картинку для превью в этой сессии.
   • Калибровка масштаба — «Отметить 2 точки на линейке»: кликни начало и
     конец отрезка линейки на карте, введи, сколько это КМ (напр. 1800).
     ВАЖНО сделать один раз — от этого зависят все расстояния.
   • ✋ Двигать — панорама + перетаскивание городов мышью.
   • 📍 Город — клик по карте добавляет город (имя + тип: capital/city/cursed).
   • 〜 Река — клик расставляет точки русла, ДВОЙНОЙ клик завершает реку.
   • Галка «Показать маску воды» — наложить маску поверх карты (синее=море,
     голубое=река), чтобы СВЕРИТЬ её с картой и поправить кистью.
   • 🖌 Море / 🖌 Суша — кисть: закрашиваешь, где вода, где земля
     (маршруты ходят только по суше/воде/реке — это та самая маска).
     Маска и реки сгенерированы автоматически из карты — это хорошая основа;
     север (снег) и редкие места можно подправить кистью.
   • Список городов — удаление крестиком.
   • ⬇ Экспорт — скачивает config.json (ВСЕ твои правки).
   • ⬆ Импорт — загрузить config.json обратно.
   • ↻ Пересоздать маску — заново сгенерировать черновик суша/море по цвету.

  Правки сохраняются автоматически в браузере. Но чтобы их увидели ИГРОКИ —
  нужно «Экспорт» → положить config.json в папку → перезалить (см. п.3).

────────────────────────────────────────────────────────────
3. КАК ОПУБЛИКОВАТЬ / ОБНОВИТЬ
────────────────────────────────────────────────────────────
  ПЕРВЫЙ РАЗ:
   1) Зайди на  app.netlify.com/drop
   2) Зарегистрируй бесплатный аккаунт (чтобы ссылка не менялась).
   3) Перетащи папку map-explorer в окно.
   4) Получишь ссылку вида  rift-map.netlify.app  — её на форум.

  ОБНОВИТЬ (новая карта / города / реки):
   1) В редакторе (#editor) поправила → «Экспорт» → скачался config.json.
   2) Замени config.json в папке (и map.webp, если новая карта).
   3) На Netlify: твой сайт → перетащи папку заново. Ссылка та же.

  СЛОИ КАРТЫ (файлы в папке):
   • Map 4k.webp        — географическая, БЕЗ подписей. Её видят игроки.
   • Map 4k admin.webp  — С подписями. Только админ-подложка (вкладка «С отметками»).

  ОБНОВИТЬ КАРТУ — АВТО-АДАПТАЦИЯ МАСКИ (без перегенерации руками):
   1) Перезапиши  Map 4k.webp  новой версией (тот же размер 4096×3072,
      тот же кадр — чтобы города совпали). Никаких переименований/копий.
   2) Открой приложение (Ctrl+Shift+R). Приложение само замечает, что карта
      другая, и пересоздаёт маску воды под неё — ничего нажимать не надо.
   3) (желательно) #editor → «Экспорт» → положи config.json в папку
      (тогда маска уже просчитана, быстрее грузится у игроков).
   4) Перезалей папку на Netlify.
   • Если Windows пишет «File in use» при замене webp — это превью-миниатюра.
     Закрой панель просмотра в Проводнике или нажми «Try Again».
   • Карту держи в ~3–6 МБ (не lossless) — иначе долго грузится.

────────────────────────────────────────────────────────────
4. ЗАМЕТКИ ПО v1 (что пока упрощено)
────────────────────────────────────────────────────────────
  • Маска суша/море авто-генерится по цвету — это ЧЕРНОВИК. Пройдись кистью.
  • Скорости путников (км/день) заданы в коде (PERSONAS) — можно поправить:
      Простолюдин: пешком 28, река ↓70 ↑16, море 85
      Купец:       верхом 50, река ↓80 ↑20, море 160
      Курьер:      115,        река ↓95 ↑25, море 250
  • Реки нужно нарисовать в редакторе — без них речные пути не считаются.
    Устье (куда течёт) определяется само: тот конец реки, что ближе к морю.
  • Паровозы Сильвании и тонкая настройка Врат — следующими версиями.
  • Время Врат — формула-приближение по аналогии с авиаперелётами.
